## Prerequisites

- Node 20+
- Cloudflare account + Wrangler CLI
- Optional: Docker (to run Tron locally)

## 1) Configure environment

Set these core variables per service:

Billing Worker:

- <code>SERVICE_SECRET_LISA</code>, <code>SERVICE_SECRET_DASHBOARD</code>
- Optional rate vars: <code>RATE_CENTS_PER_100K_AI_INPUT</code>, <code>RATE_CENTS_PER_100K_AI_OUTPUT</code>

## 2) Deploy Workers

From each folder:

```bash
wrangler deploy
```

## 3) Run Tron (choose one)

- Use hosted: <code>TRON_API_BASE=https://tron.tera.space</code> and Bearer <code>123456</code>
- Or local: in <code>Roadrunner-Tron</code> run Docker compose and set <code>TRON_API_BASE</code> accordingly

## 4) Trigger an agent

Dashboard calls Lisa which calls Tron.

```bash
curl -X POST "$LISA_BASE/v1/consumer/agents/trigger" \\
  -H "Authorization: Bearer $CONSUMER_SECRET" \\
  -H "content-type: application/json" \\
  -d '{"agent":"glitter","prompt":"go to google","stepLimit":20}'
```

Poll the run:

```bash
curl -H "Authorization: Bearer $CONSUMER_SECRET" \\
  "$LISA_BASE/v1/consumer/agents/runs/$RUN_ID"
```

Stop the browser session:

```bash
curl -X DELETE -H "Authorization: Bearer $CONSUMER_SECRET" \\
  "$LISA_BASE/v1/consumer/session?sessionId=$TAURINE_SESSION_ID"
```

<Tip>
  The Dashboard already proxies these calls via <code>/api/agents/*</code>.
</Tip>
