## Get an API key

- Copy your Agent API key from the dashboard.
- Send it in the header: `Authorization: Bearer <AGENT_API_KEY>`.

Base URL: `https://tron.tera.space`

You can optionally set environment variables in your shell:

- `API_BASE=https://tron.tera.space`
- `API_KEY=<your key>`

## Create a task

<CodeGroup>

```bash cURL
curl -X POST "$API_BASE/v1/task/create" \
  -H "Authorization: Bearer $API_KEY" \
  -H "content-type: application/json" \
  -d '{"agent":"glitter","prompt":"What does example.com's homepage say?","mode":"text","stepLimit":10}'
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')

resp = requests.post(
    f"{API_BASE}/v1/task/create",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "content-type": "application/json",
    },
    json={"agent": "glitter", "prompt": "What does example.com's homepage say?", "mode": "text", "stepLimit": 10},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;

const res = await fetch(`${API_BASE}/v1/task/create`, {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${API_KEY}`,
    'content-type': 'application/json',
  },
  body: JSON.stringify({ agent: 'glitter', prompt: "What does example.com's homepage say?", mode: 'text', stepLimit: 10 }),
});
console.log(await res.json());
```

</CodeGroup>

Example response (200):

```json
{ "taskId": "02a94d68-b9e9-49bd-a3e4-54b8426e3550" }
```

## Get a task

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer $API_KEY" \
  "$API_BASE/v1/task/$TASK_ID"
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')
TASK_ID = os.getenv('TASK_ID')

resp = requests.get(
    f"{API_BASE}/v1/task/{TASK_ID}",
    headers={"Authorization": f"Bearer {API_KEY}"},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;
const TASK_ID = process.env.TASK_ID!;

const res = await fetch(`${API_BASE}/v1/task/${TASK_ID}`, {
  headers: { Authorization: `Bearer ${API_KEY}` },
});
console.log(await res.json());
```

</CodeGroup>

Typical fields include a `status` (e.g., queued, running, completed, failed) and implementation-specific details like `result` once complete.

## List tasks

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer $API_KEY" \
  "$API_BASE/v1/task/list?pageSize=20&page=1"
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')

resp = requests.get(
    f"{API_BASE}/v1/task/list",
    headers={"Authorization": f"Bearer {API_KEY}"},
    params={"pageSize": 20, "page": 1},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;

const res = await fetch(`${API_BASE}/v1/task/list?pageSize=20&page=1`, {
  headers: { Authorization: `Bearer ${API_KEY}` },
});
console.log(await res.json());
```

</CodeGroup>

Learn more in Guides: [/agent-api/guides/agent-tasks](/agent-api/guides/agent-tasks)

Tip: Prefer environment variables over hardcoding keys. In PowerShell, set an env var for the current session with:

```powershell
$env:API_BASE = "https://tron.tera.space"
$env:API_KEY = "<your key>"
```
