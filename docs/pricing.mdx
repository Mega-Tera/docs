---
title: "Pricing"
description: "Simple usage-based pricing for Browser and Agent APIs"
---


| Category      | Browser API                      | Agent API                                          |
| :------------ | :------------------------------- | :------------------------------------------------- |
| Latency       | &lt;3s boot time                 | &lt;5s boot time                                   |
| Rate Limits   | 10 concurrent sessions (message us to increase)        | 10 concurrent sessions (message us to increase)                        |
| Session Limit | 1 hr                             | 1 hr (or a configurable step limit)                |
| Pricing       | $0.09 / hr (per browser session, minimum 1 minute bill time) | `$0.5` per million input tokens, `$2` per million output tokens.   |

Notes

- Token metering occurs after a run completes
- Browser sessions are billed at a 1 minute minimum
