## What is an agent task?

An agent task asks a named agent (e.g., `glitter`) to perform work based on your prompt. You create a task, then check its status or results, and you can list recent tasks for auditing.

Base URL: `https://tron.tera.space`

Authentication: Send `Authorization: Bearer <AGENT_API_KEY>`.

## Create a task

<CodeGroup>

```bash cURL
curl -X POST "$API_BASE/v1/task/create" \
  -H "Authorization: Bearer $API_KEY" \
  -H "content-type: application/json" \
  -d '{"agent":"glitter","prompt":"What does example.com's homepage say?","mode":"text","stepLimit":10}'
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')

resp = requests.post(
    f"{API_BASE}/v1/task/create",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "content-type": "application/json",
    },
    json={"agent": "glitter", "prompt": "What does example.com's homepage say?", "mode": "text", "stepLimit": 10},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;

const res = await fetch(`${API_BASE}/v1/task/create`, {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${API_KEY}`,
    'content-type': 'application/json',
  },
  body: JSON.stringify({ agent: 'glitter', prompt: "What does example.com's homepage say?", mode: 'text', stepLimit: 10 }),
});
console.log(await res.json());
```

</CodeGroup>

Example response (200):

```json
{ "taskId": "02a94d68-b9e9-49bd-a3e4-54b8426e3550" }
```

## Get a task

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer $API_KEY" \
  "$API_BASE/v1/task/$TASK_ID"
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')
TASK_ID = os.getenv('TASK_ID')

resp = requests.get(
    f"{API_BASE}/v1/task/{TASK_ID}",
    headers={"Authorization": f"Bearer {API_KEY}"},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;
const TASK_ID = process.env.TASK_ID!;

const res = await fetch(`${API_BASE}/v1/task/${TASK_ID}`, {
  headers: { Authorization: `Bearer ${API_KEY}` },
});
console.log(await res.json());
```

</CodeGroup>

Expect fields like `status` (queued, running, completed, failed) and, once completed, an optional `result` payload provided by the agent.

## List tasks

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer $API_KEY" \
  "$API_BASE/v1/task/list?pageSize=20&page=1"
```

```python Python
import os, requests

API_BASE = os.getenv('API_BASE')
API_KEY = os.getenv('API_KEY')

resp = requests.get(
    f"{API_BASE}/v1/task/list",
    headers={"Authorization": f"Bearer {API_KEY}"},
    params={"pageSize": 20, "page": 1},
)
print(resp.json())
```

```typescript TypeScript
const API_BASE = process.env.API_BASE!;
const API_KEY = process.env.API_KEY!;

const res = await fetch(`${API_BASE}/v1/task/list?pageSize=20&page=1`, {
  headers: { Authorization: `Bearer ${API_KEY}` },
});
console.log(await res.json());
```

</CodeGroup>

## Notes

- `stepLimit` caps how many autonomous steps the agent can take.
- Use pagination on list with `pageSize` and `page`.
- Keep your API key secret. Prefer environment variables (e.g., PowerShell: `$env:API_BASE = "https://tron.tera.space"; $env:API_KEY = "<your key>"`).
