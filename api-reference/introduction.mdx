## Agent API

`POST /v1/consumer/agents/trigger`

Request:

```json
{ "agent": "glitter", "prompt": "go to google", "stepLimit": 20 }
```

Response (202):

```json
{ "runId": "...", "cdpUrl": "wss://...", "taurineSessionId": "..." }
```

- `GET /v1/consumer/agents/runs/:id` — returns run state and usage

- `DELETE /v1/consumer/session?sessionId=...` — stop the Taurine session

Auth: `Authorization: Bearer <API_KEY>`

## Browser API

- `POST /v1/consumer/session` — start a browser
- `GET /v1/consumer/session?sessionId=...` — read status
- `DELETE /v1/consumer/session?sessionId=...` — stop the browser
- `GET /v1/consumer/session/list?page=&pageSize=` — list sessions (paginated)
