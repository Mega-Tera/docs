## Dashboard server routes (proxies)

- `POST /api/agents/trigger` → Lisa `POST /v1/consumer/agents/trigger`
- `GET /api/agents/runs/:id` → Lisa `GET /v1/consumer/agents/runs/:id`

Headers:

```
x-bc-consumer: <consumerSecret>
content-type: application/json
```

## Lisa (Cloudflare Worker)

- `POST /v1/consumer/agents/trigger`

Request:

```json
{ "agent": "glitter", "prompt": "go to google", "stepLimit": 20 }
```

Response (202):

```json
{ "runId": "...", "cdpUrl": "wss://...", "taurineSessionId": "..." }
```

- `GET /v1/consumer/agents/runs/:id` — returns Tron JSON for the run

- `DELETE /v1/consumer/session?sessionId=...` — stop the Taurine session

Auth: `Authorization: Bearer <consumerSecret>` (from Dashboard exchange).

## Tron (Agents)

- `POST /v1/task/create`

```bash
curl -X POST "$TRON/v1/task/create" \\
  -H "Authorization: Bearer 123456" -H "content-type: application/json" \\
  -d '{"agent":"glitter","prompt":"go to google","stepLimit":20}'
```

- `GET /v1/task/:id`

```bash
curl -H "Authorization: Bearer 123456" "$TRON/v1/task/<id>"
```

Returns:

```json
{
  "id": "...",
  "task": "agent",
  "state": "completed",
  "inputTokens": 8048,
  "outputTokens": 1005,
  "data": { "cdpUrl": "wss://..." },
  "result": { "answer": "..." }
}
```
